** Portfolio **

Languages used : angular and typescript.

1. This project is all about the portifolio of my profile.

2. This project is based on angualr and screenshots are included for the reference.

3. Run `ng serve` for a development server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

4. A login page appears, where the user has to fill the details to enter inside the portfolio.

5. If the user is not registered a signup registration page appears where the user has to fill all the details and then user can login.

6. Run `ng generate component component-name` to generate a new component.

7. Run `ng build` to build the project.
